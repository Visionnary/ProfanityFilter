Settings

Filter Settings

There are three modes for the filter to operate in, each with different settings to fine-tune your experience.

Full Censor Mode

In Full Censor mode, the offensive words are completely censored and displayed as asterisks.

Partial Censor Mode

In partial censor mode, the offensive words will have their first and last letters shown while everything else is censored.

Off Mode

This mode disables filtering and is useful for detecting words without modifying the page.

Lists

Whitelist

The Word Whitelist provides a mechanism to list words/phrases that should NOT be filtered. Whitelist entries can be case sensitive (case matters). For example, if you added the word "dog" to the filter, but don't want "Dog" (capitalized) to be filtered, you could add "Dog" as a case-sensitive whitelisted word and it will no longer be filtered.
